# Matrices

Nested arrays, or matrices, are key data structures in graphics rendering
and modern life more broadly. Many of the devices we use today &mdash;
computer monitors, phone screens, smart watch displays, LCDs &mdash; are
applications of a pixel matrix, each pixel turning on or off, 1 or 0.
